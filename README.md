# 🍔 Food Ordering Website

## 📌 Project Description
This is a responsive Food Ordering Website developed using HTML, CSS, and JavaScript. Users can create a new account, sign in, search for their favorite food items, and place food orders through a simple and user-friendly interface.

## ✨ Features
- User Registration (Sign Up)
- User Login (Sign In)
- Search Food Items
- Place Food Orders
- Order Confirmation Message
- Responsive User Interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How to Run
1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Run the project using Live Server or open `index.html` in your browser.
4. Sign up if you are a new user.
5. Sign in and search for your favorite food.
6. Place your order and receive an **"Order Successfully"** message.
